# 🚲 Bikcraft - Bicicletas Elétricas

O **Bikcraft** é um site institucional desenvolvido para uma empresa fictícia que comercializa bicicletas elétricas artesanais. 
Este projeto foi o ponto central do curso de **HTML e CSS para Iniciantes** da [Origamid](https://www.origamid.com/), focado em transformar um design do Figma em um código real, semântico e responsivo.

---

## 🚀 Sobre o Projeto

O objetivo foi construir uma interface moderna que prioriza a experiência do usuário (UX) e a performance. 
O site inclui páginas de produtos, seguros, contato e um formulário de orçamento funcional (estilização e validação).

### 🧠 O que eu pratiquei:
- **HTML5 Semântico:** Estruturação para melhor acessibilidade e SEO.
- **CSS Grid & Flexbox:** Layouts complexos e alinhamentos precisos.
- **Responsividade:** Estratégia Mobile-First para garantir que o site funcione em qualquer tela.
- **Componentização:** Criação de componentes reutilizáveis como botões, listas e tipografia.
- **JavaScript:** Manipulações simples para interatividade (menu mobile, perguntas frequentes).
